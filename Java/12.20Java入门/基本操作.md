# Java基础操作

### 注释

> 单行注释:	
>
> ```java
> //...
> //这是一行注释
> ```
>
> 多行注释:	
>
> ```java
> /*...*/
> /*这是多行注释
> **这是多行注释*/
> ```
>
> 文档注释
>
> ```java
> /**...*/
> ```

### 缩进

> 为了便与阅读,编写代码时候会使用缩进,缩进都懂吧...不解释了,一般缩进用4个空格
>
> ```java
> public class HelloWorld{
>     public static void main(String[] args){
>         System.out.println("Hello World");
>     }
> }
> ```
